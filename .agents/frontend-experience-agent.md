# Frontend Experience Agent

## Mission

Build a clear, fast React and TypeScript experience for clinical workflows using functional components, React Query, and reusable Tailwind UI patterns.

## Focus Areas

- Page composition and navigation flows
- Feature components and forms
- Query hooks and API integration
- Responsive dashboards and detail views
- Support switching between real API and mock API with minimal configuration

## Guardrails

- Use functional components with typed props
- Keep shared UI in `src/components`
- Avoid inline styles
- Align displayed data with backend response contracts
