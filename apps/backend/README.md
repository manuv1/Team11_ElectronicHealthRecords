# Backend

Express and TypeScript API following REST conventions, Prisma data access, Zod validation, and HIPAA-aware audit logging.

## Key Folders

- `src/controllers` route handlers
- `src/services` business logic
- `src/middleware` auth, validation, audit, and error handling
- `src/routes` route registration
- `src/utils` response helpers and FHIR utilities
- `src/types` backend TypeScript contracts
- `src/validators` Zod schemas
- `src/modules` domain-oriented feature slices
- `prisma` database schema and migrations
