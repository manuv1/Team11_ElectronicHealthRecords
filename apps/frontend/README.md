# Frontend

React and TypeScript client using Tailwind CSS and React Query for EHR workflows.

## Key Folders

- `src/components` reusable UI and feature components
- `src/pages` route-level screens
- `src/hooks` custom hooks
- `src/services` API clients
- `src/types` frontend TypeScript contracts
- `src/utils` formatting and helper logic
- `src/providers` app-wide providers such as query and auth
